Introduction

Welcome to the API documentation. This page provides an overview of the available API endpoints and how to use them.

Face Detection and Cropping

This API endpoint allows you to detect and crop faces from a provided image. The input image must be in JPEG format and should have a minimum width and height of 200 pixels. The response will include the number of detected faces and their corresponding cropped images encoded in base64 format.

API Endpoints

POST /tools/face-detection-and-split/

Detect faces within the provided image and return the cropped faces and original inage with face detect.

                    
curl -X 'POST' \
    'http://127.0.0.1/tools/face-detection-and-split/' \
    -H 'accept: application/json' \
    -H 'Content-Type: multipart/form-data' \
    -F 'face_image=@1.jpg;type=image/jpeg'

Response:

                    
{
    "original_image": "/9j/4AAQSkZJRgABAQAASABIAAD...",
    "total_face_count": 3,
    "cropped_image_list": [
        "/9j/4AAQSkZJRgABAQAASABIAAD...",
        "/9j/4AAQSkZJRgABAQAASABIAAD...",
        "/9j/4AAQSkZJRgABAQAASABIAAD..."
    ]
}
